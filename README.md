# pullword-java

pullword java client （non-official）  

[pullword](http://pullword.com/)是由[梁斌博士](http://weibo.com/pennyliang)开发的一款优秀的在线中文分词服务  
本客户端实现了其http调用的封装 

1.getXXX方法为采用get方式调用api  
2.postXXX方法为采用post方式调用api  
3.XXXPlainText方法只返回分词后的词，以空格分隔，不包括probability  
4.各方法参数含义与[pullword官方接口](http://api.pullword.com/)定义保持一致
